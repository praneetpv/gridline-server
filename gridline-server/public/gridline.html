Close a ring (interlink line)

Connects two existing breakers — often across feeders — to close a ring. No new node is created.

Leave unchecked for the normal case in India — the ring is maintained by operating the AB switches at each end, with no intermediate switch on the interlink itself. All interlinks load Closed; check this only for the rare interlink that really does have its own switch, so it can be opened/closed independently.
